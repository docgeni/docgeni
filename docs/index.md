 ## 快速上手
手动创建第一篇文档

```bash
// 创建组件开发的目录
$ mkdir library && cd library

// 安装 Docgeni CLI
$ npm i @docgeni/cli

// 创建文档
$ mkdir docs && echo '# Hello Docgeni!' > docs/index.md

// 预览文档
$ npx @docgeni/cli start
```
